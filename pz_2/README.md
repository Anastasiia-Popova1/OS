# Відповіді на контрольні запитання

1. Що таке масив? Дайте визначення.
Масив - це структура даних, яка зберігає фіксовану кількість елементів одного типу в послідовних комірках пам'яті.

2. В чому переваги представлення інформації у вигляді масиву?
Швидкий доступ до будь-якого елемента за індексом, компактне зберігання в пам'яті, зручність обробки однотипних даних у циклах.

3. Яким чином визначається довжина масиву при його ініціалізації?
Довжина визначається кількістю елементів у списку ініціалізації, або явно задається в квадратних дужках.

4. Що таке вказівник? Навіщо він використовується?
Вказівник - це змінна, яка зберігає адресу іншої змінної в пам'яті. Використовується для динамічного виділення пам'яті, роботи з масивами, передачі параметрів за адресою.

5. Як оголошується змінна типу вказівник?
тип *ім'я_змінної.

6. Чому при оголошенні вказівника необхідно вказувати тип змінної, яка адресується з його допомогою?
Щоб компілятор знав, скільки байтів займає змінна і як інтерпретувати дані за цією адресою.

7. Розкажіть про посилання.
Посилання - це псевдонім для вже існуючої змінної. Воно не займає додаткової пам'яті і не може бути переналаштоване на іншу змінну.

8. Навіщо використовуються змінні типу-посилання?
Для уникнення копіювання великих об'єктів, для зміни аргументів функції всередині неї, для перевантаження операторів.

*9. Яке значення операції & ?
& - операція взяття адреси.
* - операція розіменування.

10. Що таке динамічний масив?
Масив, пам'ять для якого виділяється під час виконання програми, а не на етапі компіляції.

11. Яке застосування операцій new і delete?
new - виділяє пам'ять динамічно. delete - звільняє раніше виділену пам'ять.

12. Як представлений рядок в С?
Як масив символів типу char, який закінчується нуль-термінатором '\0'.

13. Назвіть основні функції роботи з рядками.

14. Наведіть приклади функції модуля string (C++).
